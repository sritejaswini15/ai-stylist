# Clueless — Your AI Stylist

Your closet, reimagined. Clueless is an AI-powered personal styling app that learns your taste, curates outfits, and transforms the way you dress — effortlessly.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router

## Getting Started

```bash
npm install
npm run dev
```
